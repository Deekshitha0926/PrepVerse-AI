# PrepVerse AI

AI-powered mock interview platform built using MERN stack and Groq API.

## Features

- AI-generated MCQs
- Coding interview round
- Monaco code editor
- Multiple programming languages
- AI evaluation & feedback
- Timer and tab-switch detection
- JWT Authentication

## Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- Groq API
- Tailwind CSS
- Monaco Editor

## Run Project

### Client

```bash
npm install
npm run dev
```

### Server

```bash
npm install
npm run server
```
